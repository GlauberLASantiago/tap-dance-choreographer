📖 Manual Completo de Uso

🎥 1. Vídeo e Treino (Loop) Selecione trechos na tabela para repetir partes específicas (Treino). Dê duplo clique nos passos da tabela para editá-los livremente.
👣 2. Registro de Passos Clique nos botões das barras laterais (Esquerda e Direita) para gravar batidas. Use o campo "Novo..." para criar passos personalizados.
📊 3. Linha do Tempo / Zoom A linha de onda de áudio indica batidas sonoras. Use o controle de Zoom Horizontal para ajuste milimétrico de tempos.
🗣️ 4. Ditado por Voz (TTS) Ative a voz no player. O app pronunciará os passos no tempo exato, priorizando vozes femininas de alta qualidade (Microsoft Online).
🔄 5. Versão Espelho Marque a opção "Gerar versão espelhada" na exportação para gerar a coreografia invertida de pés (E ➔ D, D ➔ E) automaticamente.
📄 6. Exportação Sem Contagem Exporte em formato de Tabela ou Textual (em PDF ou copiado para colar onde quiser) contendo todas as opções de agrupamentos de combinações.
